# Send a SightTrack growth report

Use this when an optometrist says **"send [patient] a SightTrack report."** It emails the parent their child's eye-growth chart, or prints a hard copy. The optom records the measurement during the consult — you just send it.

## Steps

1. On the Hub home screen, tap the **SightTrack Growth Tracker** tile. It opens in a new tab.
2. In the **Find patient — Optomate** box, type the child's **surname** (or first name).
3. Tap the right patient in the list — **check the name AND date of birth match** before going on.
4. A growth chart appears with dots on it — that's the child's eye-growth.

> IF no chart appears, or it says there are no readings: the optometrist hasn't recorded the measurement yet. Do NOT send — let the optom know so they can add it.

5. **To email the parent:** tap **Email latest report to parent**. Check the email address shown, tap OK, and it sends. Done.
6. **To print a hard copy:** tap **Print report (hard copy)**, then **Print / Save as PDF**.

> The email goes to the address on the patient's Optomate file. If it says there's no email, ask the patient for one and add it in Optomate first.

> A copy of every report is saved automatically into the patient's Optomate documents — you don't need to file it anywhere.

> Always double-check the name and date of birth. Never send a report for a different child than the optom asked for.
